## 目标
- 在 React 同步页面新增“分店订货单主表”“分店订货单详情”“分店订货单（主+详情）一键同步”的按钮与结果展示。

## 变更点
- services：在 `ReactUmi/my-app/src/services/dataSync.ts` 新增：
  - `syncWareHouseOrders()` → POST `/api/react-sync/warehouse-orders`
  - `syncWareHouseOrderDetails()` → POST `/api/react-sync/warehouse-order-details`
  - `syncWareHouseOrdersAll()` → POST `/api/react-sync/warehouse-orders-all`
- 页面：在 `SystemSettings/HqDataSync/index.tsx` 引入上述方法，新增三张紧凑卡片与按钮，展示 `SyncResult`（状态/新增/错误/耗时/消息）。

## 验证
- 前端构建通过（`npm run build`）。
- 点击按钮触发同步端点，页面提示与结果展示正常。

## 影响范围
- 仅前端新增服务方法与页面卡片，不影响其他功能。
