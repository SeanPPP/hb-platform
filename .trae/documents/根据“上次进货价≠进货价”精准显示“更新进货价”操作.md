## 规则更新
- 有商品（exists=true）时：
  - last = meta?.supplierDetect?.currentPurchasePrice ?? r.lastPurchasePrice ?? 0
  - now = r.purchasePrice ?? 0
  - 若 last>0 && now>0：
    - 差异≥1.5倍（now >= 1.5*last 或 last >= 1.5*now）→ 返回 "待操作"
    - 差异未达1.5倍但 last !== now → 返回 "更新进货价"
    - 否则 → 按条码匹配保持原逻辑（不操作/待操作）
- 无商品（exists=false）：仍按“新建商品/待操作”原逻辑。
