## 修改 hasRole 函数为不区分大小写比较

### 修改文件

**文件：** `src/access.ts`

**修改位置：** 第45-47行

**修改前：**

```typescript
const hasRole = (role: string): boolean => {
  return currentUser.roleNames?.includes(role) || false;
};
```

**修改后：**

```typescript
const hasRole = (role: string): boolean => {
  if (!currentUser.roleNames) return false;
  return currentUser.roleNames.some(r => 
    r.toLowerCase() === role.toLowerCase()
  );
};
```

### 修改说明

1. 使用 `some()` 方法替代 `includes()`
2. 将角色名转换为小写后比较
3. 添加空值检查，避免 `undefined` 错误

### 预期效果

✅ 不区分大小写：`WarehouseManager`、`warehousemanager`、`WAREHOUSEMANAGER` 都能匹配
✅ 仓库管理员（`WareHouseManger`）可以正常访问权限
✅ "国内采购"导航正常显示
✅ 所有角色检查都支持大小写不敏感
