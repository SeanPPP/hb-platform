我将通过添加样式来显示语言切换按钮。

**原因**：
虽然代码中已经引入了 `<SelectLang />` 组件，但对应的 CSS 类 `.lang` 在样式文件 `index.less` 中尚未定义，导致按钮位置不正确或被遮挡。

**实施计划**：
修改 `d:\Development\cline\blazor\ReactUmi\my-app\src\pages\Login\index.less`：
1.  为 `.container` 添加 `position: relative`，确保绝对定位的子元素相对于容器定位。
2.  添加 `.lang` 类，将语言切换按钮绝对定位到页面右上角。

**样式代码预览**：
```less
.container {
  // ... existing styles
  position: relative; // 新增

  .lang {
    position: absolute;
    top: 24px;
    right: 24px;
    z-index: 100;
    
    // 调整图标颜色和背景，使其更明显
    :global {
      .ant-dropdown-trigger {
        font-size: 24px;
        padding: 4px;
        border-radius: 4px;
        color: rgba(0, 0, 0, 0.65);
        
        &:hover {
          background-color: rgba(0, 0, 0, 0.05);
        }
      }
    }
  }
}
```
这将在页面右上角显示通用的语言切换图标（地球图标）。
