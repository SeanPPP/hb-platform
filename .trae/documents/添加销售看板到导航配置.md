## 添加销售看板到导航配置

### 需要修改的文件

#### 1. KeepAliveTabLayout.tsx
- 在 `componentMap` 中添加 `SalesDashboard` 组件映射
- 位置：第 125 行 `UserProfile: React.lazy(() => import('@/pages/UserProfile') as any),` 之前添加

```typescript
SalesDashboard: React.lazy(() => import('@/pages/SalesDashboard') as any),
```

#### 2. app.tsx
- 在 `iconMap` 中添加 `DashboardOutlined` 图标映射
- 位置：第 22 行导入 `DashboardOutlined` 图标
- 位置：第 147 行 `iconMap` 中添加

```typescript
import { DashboardOutlined, ... } from '@ant-design/icons';
```

```typescript
const iconMap: Record<string, React.ReactNode> = {
  ...
  DashboardOutlined: <DashboardOutlined />,
  ...
};
```

### 修改后的效果
- 用户可以在侧边栏看到"销售看板"菜单项
- 点击菜单项后会在 Tab 中打开销售看板页面
- 图标正常显示为仪表盘图标
