**目标**

* 主内容区域固定在可视范围内（不随内部内容整体增长）

* 底部元素始终与页面底部对齐（容器内部滚动，底部不抖动）

**实施范围**

* 统一优化 `WarehouseCategories` 与 `WarehouseProducts` 两页的左右分栏与右侧内容区布局

**优化方案**

* 外层容器（split-wrapper）：

  * 维持 `overflow: hidden` 与 `display: flex`，高度使用视口差值：保持 `height: paneHeight`，由 `useLayoutEffect` 计算 `paneHeight = max(300, vh - top)`

* 左侧卡片：

  * `height: 100%`，内部容器 `flex: 1 1 auto; minHeight: 0; overflow: auto`，树在自身容器内滚动

* 右侧卡片：

  * 使用 `flex` 列布局：`height: 100%; display: flex; flexDirection: column; overflow: hidden`

  * 顶部工具条 `position: sticky; top: 0; zIndex: 10; background: '#fff'`

  * 中间表格容器：`flex: 1 1 auto; minHeight: 0; overflow: auto`

  * 底部分页容器：`flex: 0 0 auto; position: sticky; bottom: 0; zIndex: 5; background: '#fff'`

* 表格滚动：

  * 分类页：保留外置分页，`scroll={{ x, y: tableScrollY }}` 由 `paneHeight - toolbar - pager` 计算，避免与底部重叠

  * 商品页：使用表格内置分页，右侧容器仍固定高度，表格内部滚动

* 移除对容器内部内容高度的耦合：

  * 分类页删除对 `paddingBottom: paginationHeight` 的依赖，分页容器改用 `sticky bottom` 保持对齐

**验证**

* 打开两页，窗口缩放与数据增减时：

  * 页面不出现外层滚动条，左右各自滚动

  * 底部分页始终贴底，表格最后一行不被遮挡

  * 顶部工具条与底部分页在滚动时保持就位
