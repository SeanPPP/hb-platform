# 用户详情显示权限 & 编辑添加重置密码 方案

## 现状分析

### 用户详情 Drawer（index.tsx L207-L270）

当前展示字段：用户名、姓名、邮箱、状态、角色（Tag）、创建/更新时间、关联分店 List。

**缺少**：用户权限（permissions）展示——`UserDetailDto` 有 `permissions?: string[]` 但没用上。

### 编辑用户 Drawer（index.tsx L272-L296）

4 个 Tab：基本信息、角色（Transfer）、分店（Transfer）、权限（Tree）。

**缺少**：重置密码按钮。后端 `POST /api/Users/guid/{guid}/reset-password` 已实现，但前端 service 和 UI 都没有。

## 步骤 1：前端 service 层新增 resetPassword

**文件**：`hbweb_rv/src/services/userService.ts`

```typescript
import type { ResetPasswordResultDto } from '../types/user'

export async function resetUserPassword(guid: string): Promise<ResetPasswordResultDto> {
  const response = await request.post<ApiResponse<ResetPasswordResultDto>>(
    `/api/Users/guid/${guid}/reset-password`
  )
  return unwrapApiData(response)
}
```

## 步骤 4：编辑 Drawer 添加重置密码按钮

**文件**：`hbweb_rv/src/pages/System/Users/index.tsx`

### 4.1 新增 state

```typescript
const [resetPwdLoading, setResetPwdLoading] = useState(false)
```

### 4.2 新增 import

```typescript
import { LockOutlined } from '@ant-design/icons'
import { resetUserPassword } from '../../../services/userService'
```

### 4.3 新增 handler

```typescript
const handleResetPassword = async () => {
  if (!editingUser) return
  setResetPwdLoading(true)
  try {
    const result = await resetUserPassword(editingUser.userGUID)
    Modal.success({
      title: t('system.users.resetPasswordSuccess'),
      content: (
        <div>
          <p>{t('system.users.newPasswordLabel')}: <Typography.Text copyable strong>{result.newPassword}</Typography.Text></p>
          <p style={{ color: '#999' }}>{t('system.users.resetPasswordWarning')}</p>
        </div>
      ),
    })
  } catch (error) {
    message.error(t('system.users.resetPasswordFailed'))
  } finally {
    setResetPwdLoading(false)
  }
}
```

### 4.4 在基本信息 Tab 底部添加按钮

```tsx
<HasPermission code={P.Users.ResetPassword}>
  <div style={{ marginTop: 16, paddingTop: 16, borderTop: '1px solid #f0f0f0' }}>
    <Popconfirm
      title={t('system.users.resetPasswordConfirm')}
      onConfirm={() => void handleResetPassword()}
    >
      <Button icon={<LockOutlined />} loading={resetPwdLoading}>
        {t('system.users.resetPassword')}
      </Button>
    </Popconfirm>
  </div>
</HasPermission>
```
