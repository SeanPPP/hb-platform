## 修改目标
1. 移除品牌名称前的 "Brand:" 提示
2. 将品牌名称字体变大，与 "TAX INVOICE" 字体大小相近（使用 titleFont）

### 实施步骤

修改 [TaxInvoiceService.cs#L158-L167](file:///d:/Development/cline/blazor/BlazorApp.Api/Services/TaxInvoiceService.cs#L158-L167)：

```csharp
// 从：
if (store != null && !string.IsNullOrEmpty(store.BrandName))
{
    document.Add(
        new Paragraph($"Brand: {store.BrandName}", bodyFont)
        {
            Alignment = Element.ALIGN_CENTER,
        }
    );
    _logger.LogInformation($"已显示 BrandName: {store.BrandName}");
}

// 改为：
if (store != null && !string.IsNullOrEmpty(store.BrandName))
{
    document.Add(
        new Paragraph(store.BrandName, titleFont)
        {
            Alignment = Element.ALIGN_CENTER,
        }
    );
    _logger.LogInformation($"已显示 BrandName: {store.BrandName}");
}
```
