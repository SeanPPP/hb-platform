# 配货单打印优化计划

为了满足配货单 Name 最多显示2行，且超过 A4 纸时自动分页并重复页头的需求，我们需要调整 CSS 样式和打印结构。

## 1. 限制商品名称行数
通过 CSS 限制 `.col-name` 的高度和行数。

**CSS 修改方案:**
*   使用 `display: -webkit-box`
*   `-webkit-line-clamp: 2`
*   `-webkit-box-orient: vertical`
*   `overflow: hidden`

## 2. 自动分页与页头重复
利用 CSS 的打印特性 `thead { display: table-header-group; }` 可以让表格在每一页打印时自动重复表头。
对于页面顶部的 "Order No", "Store" 等信息，如果希望每页都显示，有两种方案：
1.  **将其放入 `<thead>`**: 将整个页头信息作为表格的第一个表头行（合并单元格），或者将页头信息放在 `<thead>` 内部的 `<tr>` 中。这是最标准的 HTML 打印分页重复表头的方法。
2.  **固定定位 (Fixed Positioning)**: 使用 `position: fixed` 将页头固定在每一页顶部，但需要处理内容遮挡问题（设置 `padding-top`）。
3.  **JavaScript 手动分页**: 像 Invoice 页面那样计算高度手动切割分页。

**推荐方案**: **将页头信息放入 `<thead>`**。
这样浏览器会自动处理分页，并且每页都会打印表头信息和列标题。

## 实施步骤

1.  **修改 `PickingList/index.less`**:
    *   添加限制 Name 行数的样式。
    *   调整打印相关的样式，确保表格跨页断行正常。

2.  **修改 `PickingList/index.tsx`**:
    *   重构 HTML 结构，将原本在 `<table>` 外部的 `.header-section` 移入 `<table>` 的 `<thead>` 中。
    *   为了保持布局，可以在 `<thead>` 中添加一个跨列的行 (`<tr><td colSpan={9}>...</td></tr>`) 来放置订单头信息。
    *   或者，保持 `.header-section` 在外部（仅第一页显示），仅让列标题 (`th`) 重复。
    *   **用户需求是 "每页都要添加 页头"**。
    *   因此，必须将 Header Info 放入 `<thead>`。

    **结构草稿:**
    ```tsx
    <table className="picking-table">
      <thead>
        {/* 第一行：订单头信息 (跨所有列) */}
        <tr className="header-info-row">
           <td colSpan={9}>
              <div className="header-section">
                 {/* Order No, Store, Date... */}
              </div>
           </td>
        </tr>
        {/* 第二行：列标题 */}
        <tr className="column-headers">
           <th>#</th>
           <th>Item No</th>
           {/* ... */}
        </tr>
      </thead>
      <tbody>
        {/* ... items ... */}
      </tbody>
    </table>
    ```

3.  **测试**: 打印预览查看分页效果和名称截断效果。
