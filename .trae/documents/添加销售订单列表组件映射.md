## 修复销售订单列表组件映射问题

### 需要修改的文件

#### KeepAliveTabLayout.tsx
在 `componentMap` 中添加 `PosmSalesOrders` 组件的动态导入：
```typescript
PosmSalesOrders: React.lazy(() => import('@/pages/PosmSalesOrders') as any),
```

这样，当用户点击"销售订单列表"菜单时：
1. `app.tsx` 中的 `pathComponentMap` 匹配到 `PosmSalesOrders` 组件名
2. `KeepAliveTabLayout.tsx` 中的 `componentMap` 加载对应的 React 组件
3. 页面在 Tab 中正常打开
