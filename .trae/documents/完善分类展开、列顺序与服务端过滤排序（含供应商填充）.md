前端

* 左侧分类树：保持默认展开一级分类（顶级节点）。

* 右侧分类选择器：改为默认展开一级分类（不全展开），通过受控的 treeExpandedKeys 管理。

* 商品表列顺序：图片、货号、条码、名称、供应商、供应商代码、国内价、进口价、贴牌价、创建日期、更新日期；为每列开启 sorter，并为文本/数字列加 filterDropdown（输入框）以触发服务端过滤。

* 表格 rowKey 改为 productCode；分页保留 100；onChange 将 filters 与 sorter 一并传给后端。

后端

* DTO 增加 CreatedAt 字段；React 服务 Select 返回 CreatedAt/UpdatedAt；排序增加 createdAt 映射。

* 供应商填充：在查询中增加对 DomesticProduct 与第二个 ChinaSupplier 的左联，通过 HBProductNo 或 Barcode 匹配，使用 SqlFunc.IsNull<string>(s.SupplierName, cs2.SupplierName) 与对应 Code 进行兜底填充。

* 过滤与排序覆盖所有列：新增对 supplierName、supplierCode、itemNumber、barcode、productName、价格、volume、createdAt、updatedAt 的过滤键映射，排序键增加 createdAt。
