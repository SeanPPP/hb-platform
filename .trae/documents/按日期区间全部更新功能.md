## 实现按日期区间全部更新功能

### 1. 扩展 `BatchStatisticsUpdateResult` 类
添加字段跟踪每种批量更新类型的结果：
```csharp
public class BatchStatisticsUpdateResult
{
    // 现有字段...
    public bool DailyUpdateSuccess { get; set; }
    public bool HourlyUpdateSuccess { get; set; }
    public bool StoreUpdateSuccess { get; set; }
    public bool SupplierUpdateSuccess { get; set; }
    public bool StoreSupplierUpdateSuccess { get; set; }
}
```

### 2. 在 `SalesStatisticsJobService` 中创建 `BatchFullRefreshByDateRange` 方法
- 接收参数：`startDate`, `endDate`, `branchCodes`?, `supplierCodes`?
- 验证日期范围（使用现有的 `ValidateDateRange`）
- 依次调用所有批量更新方法：
  - `BatchUpdateDailyStatistics`
  - `BatchUpdateHourlyStatistics` (更新所有24小时)
  - `BatchUpdateStoreStatistics`
  - `BatchUpdateSupplierStatistics`
  - `BatchUpdateStoreSupplierStatistics`
- 汇总所有结果到统一的 `BatchStatisticsUpdateResult`
- 返回汇总结果

### 3. 在 `TaskType` 中添加新任务类型
```csharp
public const string BatchFullRefreshByDateRange = "BatchFullRefreshByDateRange";
```

### 4. 在 `StatisticsJobTriggerController` 中添加 API 端点
- `POST /api/StatisticsJobTrigger/batch-full-refresh-by-date-range`
- 请求体：`BatchFullRefreshByDateRangeRequest`
  - `startDate`, `endDate`
  - 可选：`branchCodes`, `supplierCodes`
