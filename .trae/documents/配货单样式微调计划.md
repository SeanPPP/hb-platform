# 配货单样式微调与逻辑优化计划 (修订版)

根据您的最新反馈，除了样式调整外，还需要隐藏 Send Qty 为 0 的显示。

## 1. 样式调整 (CSS)

*   **Location 不换行**:
    *   在 `.col-location` 中添加 `white-space: nowrap`。
*   **Name 列头高度**:
    *   确保 Name 列的 *内容* (`td`) 限制为2行，但 *表头* (`th`) 不受影响。
    *   方案：在 `index.tsx` 中将商品名称包裹在 `<div className="name-wrapper">` 中，并将行数限制样式应用到该 `div`，而不是 `td`。

## 2. 逻辑调整 (Logic)

*   **Send Qty 隐藏 0**:
    *   修改 `index.tsx` 中的渲染逻辑。
    *   如果 `item.allocQuantity` (即 Send Qty) 为 0 或 null/undefined，则显示为空字符串或 `-`（根据您的偏好，通常为空白更清爽，或者 `-` 表示无）。
    *   *用户指示*: "Send Qty 是0 不显示"。
    *   *实施*: 如果值为 0，渲染 `null` 或空字符串。

## 实施步骤

1.  **修改 `PickingList/index.tsx`**:
    *   商品名称单元格：包裹 `<div className="name-wrapper">`。
    *   Send Qty 单元格：添加逻辑 `(item.allocQuantity && item.allocQuantity > 0) ? item.allocQuantity : ''`。
2.  **修改 `PickingList/index.less`**:
    *   `.col-location`: 添加 `white-space: nowrap`。
    *   `.col-name`: 移除之前的 `line-clamp` 样式。
    *   添加 `.name-wrapper` 样式：应用 `-webkit-line-clamp: 2` 等限制。
