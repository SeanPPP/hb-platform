## 修改表格行高 - 固定行高

**修改内容：**
在 Table 组件中添加 `rowHeight` 属性，固定表格行高

**实现方式：**
- 添加 `rowHeight={60}`（图片高度48px + 间距）
- 确保行高固定，不随图片加载变化

**代码修改：**
```typescript
<Table<LocalSupplierInvoiceItemDto>
  ...
  scroll={{ x: 1400, y: 'calc(100vh - 380px)' }}
  pagination={false}
  rowHeight={60}
  size="small"
  bordered
/>
```
