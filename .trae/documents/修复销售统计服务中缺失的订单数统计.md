# 修复计划：添加缺失的订单数统计

## 需要修复的文件和方法

### 1. SalesStatisticsJobService.cs

#### 1.1 UpdateSupplierStatistics 方法（两个位置）
**位置**: 第682-912行 和 第1164-1378行的私有版本

**修改内容**:
- 在查询的 Select 中添加 `OrderCount = SqlFunc.AggregateCount(o.OrderGuid)`
- 在聚合中添加订单数统计
- 在更新语句中包含 `OrderCount` 字段

#### 1.2 UpdateStoreSupplierStatistics 方法（两个位置）
**位置**: 第1501-1712行 和 第2592-2806行的私有版本

**修改内容**:
- 在 GroupBy 后的 Select 中添加 `OrderCount = g.Select(d => d.B销售单号).Distinct().Count()`
- 在 `StoreSupplierSalesDetail` 创建时添加 `OrderCount` 字段
- 在更新语句中包含 `OrderCount` 字段

### 2. HBSalesRecordStatisticsService.cs

#### 2.4 ProcessStoreSupplierStatistics 方法

## 预计影响
- 修复后所有统计表都将包含正确的订单数统计
- 供应商统计和门店供应商统计将新增订单数维度
- 分时统计和分店统计将恢复订单数字段
