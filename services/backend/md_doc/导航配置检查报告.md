## 🔧 **SqlSugar导航配置规则**

### 1. **一对多关系**
```csharp
[Navigate(NavigateType.OneToMany, "子表外键字段", "当前表主键字段")]
public List<子表类型> 子表集合 { get; set; }
```

### 2. **多对多关系**
```csharp
[Navigate(typeof(中间表类型), "中间表外键字段", "当前表主键字段")]
public List<目标表类型> 目标表集合 { get; set; }
```

### 3. **一对一关系**
```csharp
[Navigate(NavigateType.OneToOne, "外键字段", "目标表主键字段")]
public 目标表类型 目标表对象 { get; set; }
```
