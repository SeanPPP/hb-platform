## 实现门店供应商销售统计功能

### 1. 在 `SalesStatisticsJobService` 中新增方法

**新增 `UpdateStoreSupplierStatistics(DateTime? date = null)` 方法**：
- 从 POSM 的 `SalesOrder` 和 `SalesOrderDetail` 联表查询
- 按 `Date + BranchCode + SupplierCode` 分组统计
- 通过 `Product.LocalSupplierCode` 判断供应商类型：
  - `LocalSupplierCode == "200"` → 从 `WarehouseProduct.DomesticProduct` 获取中国供应商信息
  - 其他 → 从 `HBLocalSupplier` 获取本地供应商信息
- 插入或更新到 `StoreSupplierSalesDetail` 表

### 2. 数据查询逻辑

```csharp
// 步骤1: 从订单明细统计基础数据
var query = _posmContext.Db
    .Queryable<SalesOrder>()
    .LeftJoin<SalesOrderDetail>((o, d) => o.OrderGuid == d.OrderGuid)
    .LeftJoin<Product>((o, d, p) => d.ProductCode == p.UUID)
    .Where(o => o.OrderTime != null && o.OrderTime.Value.Date == targetDate)
    .GroupBy((o, d, p) => new { 
        Date = o.OrderTime.Value.Date, 
        BranchCode = o.BranchCode,
        LocalSupplierCode = p.LocalSupplierCode 
    })
    .Select((o, d, p) => new {
        Date = o.OrderTime.Value.Date,
        BranchCode = o.BranchCode,
        LocalSupplierCode = p.LocalSupplierCode,
        TotalAmount = SqlFunc.AggregateSum(d.ActualAmount),
        TotalQuantity = SqlFunc.AggregateSum(d.Quantity ?? 0m)
    })
    .ToListAsync();
```

### 3. 供应商信息获取

```csharp
// 对每条统计记录：
if (data.LocalSupplierCode == "200") {
    // 获取中国供应商信息
    var warehouseProduct = await _context.Db
        .Queryable<WarehouseProduct>()
        .Include(wp => wp.DomesticProduct)
        .FirstAsync(wp => wp.ProductCode == productCode);
    
    supplierCode = warehouseProduct.DomesticProduct.SupplierCode;
    supplierName = chinaSupplier.SupplierName;
    isDomestic = true;
} else {
    // 获取本地供应商信息
    var localSupplier = await _context.HBLocalSupplierDb
        .FirstAsync(ls => ls.LocalSupplierCode == data.LocalSupplierCode);
    
    supplierCode = localSupplier.LocalSupplierCode;
    supplierName = localSupplier.Name;
    isDomestic = false;
}
```
