# 配货单显示备注信息计划

根据您的要求，我们需要在配货单打印页面显示订单的备注信息 (Remarks)。

## 1. 确认数据源
查看 `StoreOrderCartDto` (DTOs/StoreOrderDtos.cs)，发现已经包含 `Remarks` 字段。
前端调用 `getOrderDetail` 获取的数据应该已经包含了这个字段。

## 2. 前端修改 (Frontend)

在 `PickingList/index.tsx` 中，将 `Remarks` 显示在页头部分。

**设计方案**:
在 `Order Date` 下方或旁边增加一行显示备注。由于备注可能较长，建议单独占一行或在信息网格中占据显著位置。考虑到页头空间，可以在 `info-grid` 下方增加一个全宽的备注显示区域。

**修改位置**:
`src/pages/StoreOrder/PickingList/index.tsx`

**修改内容**:
在 `.header-section` 内部增加备注显示。

```tsx
<div className="header-section" style={{ ... }}>
  <div className="title">...</div>
  <div className="info-grid">...</div>
  
  {/* 新增备注显示 */}
  {order.remarks && (
    <div className="remarks-section" style={{ marginTop: 10, fontSize: 14, fontStyle: 'italic' }}>
      <span style={{ fontWeight: 'bold' }}>Remarks: </span> {order.remarks}
    </div>
  )}
</div>
```

**注意**:
由于我们将 Header Info 放入了 `<thead>` 的 `tr > td` 中，这个增加的内容也会自动在每一页重复显示，这符合通常的需求（备注是订单级的重要信息）。
