## 修改方案

### 1. 为"分店名称"和"分店代码"列添加排序功能

在 `index.tsx` 的列定义中添加 `sorter` 属性：

```typescript
// 分店名称列
{
  title: '分店名称',
  dataIndex: 'storeName',
  key: 'storeName',
  fixed: 'left',
  width: 200,
  ellipsis: true,
  sorter: true,  // 添加排序功能
}

// 分店代码列
{
  title: '分店代码',
  dataIndex: 'storeCode',
  key: 'storeCode',
  width: 150,
  sorter: true,  // 添加排序功能
}
```

### 2. 取消分页

将 `ProTable` 的 `pagination` 配置从启用分页改为禁用分页：

```typescript
// 修改前
pagination={{
  defaultPageSize: 20,
  pageSizeOptions: ['10', '20', '50', '100'],
  showSizeChanger: true,
  showQuickJumper: true,
  showTotal: (total) => `共 ${total} 条记录`,
}}

// 修改后
pagination={false}
```

或者如果要保留分页配置但默认不分页：
```typescript
pagination={false}
```
