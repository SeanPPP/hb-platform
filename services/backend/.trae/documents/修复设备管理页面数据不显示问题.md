## 修复计划

### 问题
设备管理页面虽然 API 请求成功，但页面不显示数据。

### 原因
后端配置了 CamelCase JSON 命名策略，返回字段为 `items` 和 `total`（小写），但前端代码使用大写字母 `Items` 和 `Total` 访问，导致数据读取失败。

### 解决方案
修改 `DeviceManagement/index.tsx` 第 243-246 行，将字段名改为小写：

```javascript
data: response.data.data?.items || [],    // Items → items
total: response.data.data?.total || 0,   // Total → total
```
