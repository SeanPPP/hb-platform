# 前端添加批量更新统计功能

## 需要修改的文件

### 1. `src/services/statisticsJob.ts` - 添加批量更新服务函数
- 添加 `BatchStoreUpdateRequest` 接口
- 添加 `BatchSupplierUpdateRequest` 接口
- 添加 `BatchDailyUpdateRequest` 接口
- 添加 `BatchHourlyUpdateRequest` 接口
- 添加 `BatchUpdateResult` 接口
- 添加 4 个新的服务函数：
  - `batchUpdateStoreStatistics()`
  - `batchUpdateSupplierStatistics()`
  - `batchUpdateDailyStatistics()`
  - `batchUpdateHourlyStatistics()`

### 2. `src/pages/StatisticsJobTrigger/index.tsx` - 更新UI组件
- 添加新的任务类型选项（批量更新）
- 将单日期选择器改为日期区间选择器
- 批量更新时显示开始日期和结束日期输入框
- 保留多分店/多供应商选择器（用于批量更新）
- 添加小时选择器（用于批量分时统计）
- 更新 `handleTrigger` 函数支持新的批量更新端点
- 添加加载状态和结果展示
- 更新任务说明 Alert（包含批量更新说明）

## UI 改进

### 新增任务类型
- **批量更新分店统计** - 支持日期区间 + 多分店选择
- **批量更新供应商统计** - 支持日期区间 + 多供应商选择
- **批量更新每日统计** - 支持日期区间
- **批量更新分时统计** - 支持日期区间 + 可选小时

### 表单字段变化
| 任务类型 | 日期选择 | 分店选择 | 供应商选择 | 小时选择 |
|---------|----------|---------|-----------|---------|
| 分店统计 | 单日期 | 多选 | - | - |
| 供应商统计 | 单日期 | - | 多选 | - |
| 每日统计 | 单日期 | - | - | - |
| 全量刷新 | 无 | - | - | - |
| **批量分店统计** | **日期区间** | **多选** | - | - |
| **批量供应商统计** | **日期区间** | - | **多选** | - |
| **批量每日统计** | **日期区间** | - | - | - |
| **批量分时统计** | **日期区间** | - | - | **可选** |
