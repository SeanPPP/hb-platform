# 修复计划：创建缺失的统计表

## 解决方案

重启 API 服务，让 `InitializeTablesIfNeeded()` 自动创建缺失的统计表。

## 执行步骤

1. **启动 API 服务**
   - 运行 `dotnet run --project BlazorApp.Api/BlazorApp.Api.csproj`
   - 启动时会自动调用 `InitializeTablesIfNeeded()` 方法
   - 该方法会检测并创建所有缺失的表

2. **验证表创建**
   - 查看控制台输出，确认 `✓ StoreSupplierSalesDetail 表创建成功` 等消息
   - 确认没有错误日志

## 预期结果

- ✅ `DailySalesStatistic` 表创建成功
- ✅ `HourlySalesStatistic` 表创建成功  
- ✅ `StoreSalesStatistic` 表创建成功
- ✅ `SupplierSalesStatistic` 表创建成功
- ✅ `StoreSupplierSalesDetail` 表创建成功

## 备选方案（如果重启无效）

如果重启后表仍未创建，可能需要：
1. 检查数据库连接是否正常
2. 检查 `appsettings.json` 中的连接字符串配置
3. 查看是否有权限问题阻止表创建
