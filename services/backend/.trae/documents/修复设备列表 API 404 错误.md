## 问题
前端调用 `/api/device-registration/by-store/1004` 返回 404，因为该端点有权限限制且不在 React 专用控制器中。

## 解决方案
修改前端使用 React 专用的 grid 端点，通过过滤参数获取设备列表。

## 实施步骤

### 1. 修改前端服务 `deviceService.ts`
将 `getDevicesByStoreCode` 函数改为调用 `/api/react/v1/device-registration/grid` 端点：
- 使用 POST 方法
- 传递分店代码作为过滤参数
- 返回设备列表数组

### 2. 验证修改
- 确保 API 调用路径正确
- 确保返回数据格式兼容（systemDeviceNumber 字段）
