## 实现计划

### 1. 扩展 `BatchStatisticsUpdateResult` 类
添加月份相关的统计字段：
- `TotalMonths` - 总月份数
- `ProcessedMonths` - 已处理月份数

### 2. 在 `SalesStatisticsJobService` 中创建新方法
- `BatchFullRefreshByMonths(string startYearMonth, string endYearMonth)` - 主方法
  - 参数格式：`yyyy-MM`（如 `2024-01`）
  - 验证月份范围合理性（建议限制最多12个月）
  - 遍历每个月份的每一天
  - 对每一天调用现有的 `UpdateDailyStatistics`, `UpdateHourlyStatistics`, `UpdateStoreStatistics`, `UpdateSupplierStatistics`, `UpdateStoreSupplierStatistics`
  - 记录失败日期，返回详细结果

- `ValidateMonthRange(DateTime startDate, DateTime endDate)` - 验证月份范围
  - 检查开始日期 <= 结束日期
  - 限制最多12个月（可配置）

### 3. 创建控制器端点（如需要）
在相关控制器中添加 API 端点，允许通过 HTTP 请求触发批量月份刷新
