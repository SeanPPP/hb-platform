**目标**

* 取消页面最外层滚动条，页面保持一屏高度且不随表格或分类区域高度变化。

* 左侧分类树与右侧表格分别在各自容器内滚动。\
  左侧分类树与右侧表格 高度保持 不随内部内容变化而改变

* 分页外置到表格下方，关闭响应式隐藏，始终显示页码与省略号。

**改动范围**

* 文件：`ReactUmi/my-app/src/pages/WarehouseCategories/index.tsx`

* 样式文件：`ReactUmi/my-app/src/pages/WarehouseCategories/index.less`（如需细化样式）

**具体实现**

1. 取消外层滚动与固定页面高度

* 修改最外层容器（`split-wrapper`）样式：由 `overflow: 'auto'` 改为 `overflow: 'hidden'`；高度保持使用现有 `paneHeight`，或直接改为 `height: 'calc(100vh - 160px)'`。

* 位置：约 `index.tsx:300`：

  * `style={{ height: paneHeight, overflow: 'hidden', display: 'flex', flexWrap: 'nowrap' }}`

1. 左右区域各自滚动

* 左侧分类树保持：`Card` 内部使用 `flex` + 树容器 `overflow: 'auto'`（已存在于约 `index.tsx:345`）。无需改动逻辑，仅确认 minHeight 为 `0`。

* 右侧表格区域容器保持：`style={{ flex: '1 1 auto', minHeight: 0, overflow: 'auto' }}`（已存在于约 `index.tsx:460`）。

1. 外置分页并关闭响应式隐藏

* 将 `Table` 的 `pagination` 改为 `false`：位置约 `index.tsx:501`。

* 在表格下方新增独立分页容器：

  * `<div ref={pagerRef} style={{ padding: '8px 16px', display: 'flex', justifyContent: 'flex-end', alignItems: 'center', background: '#fff' }}>`

  * `<Pagination current={page} pageSize={pageSize} total={total} onChange={(p, ps) => { setPage(p); setPageSize(ps); }} showSizeChanger responsive={false} />`

* 在文件顶部引入：`import { Pagination } from 'antd'`

* 新增 `pagerRef`：`const pagerRef = useRef<HTMLDivElement | null>(null);`

1. 精确计算表格滚动高度，避免与分页重叠

* 移除原来从 `tableWrapperRef` 查询 `.ant-table-pagination` 的方式（约 `index.tsx:282–296`）。

* 改为基于容器与工具栏、分页容器进行高度计算：

  * `const containerH = wrapperRef.current?.clientHeight || window.innerHeight;`

  * `const tbarH = toolbarRef.current?.offsetHeight || 56;`

  * `const pagerH = pagerRef.current?.offsetHeight || 0;`

  * `const available = Math.max(200, containerH - tbarH - pagerH - 8);`

  * `setTableScrollY(available);`

* 在 `useLayoutEffect` 中监听窗口尺寸与 `page/pageSize/total` 变化时重算。

1. 维持页面高度不随数据和分类区域变化

* 使用 `overflow: hidden` 的外层固定高度与内部 `minHeight: 0` 的 Flex 子容器，确保无论分类树展开、表格数据多少，页面高度保持稳定。

**验证步骤**

* 打开仓库商品管理页面：确认浏览器右侧无页面级滚动条。

* 左侧分类树可滚动；右侧表格固定表头，内部滚动；最后一行不被遮挡。

* 分页显示在底部，`•••` 省略号始终可见；切换页码和每页条数时，表格高度稳定且不与分页重叠。

**可选增强**

* 在分页容器加 `showTotal={(t)=>\`共 ${t} 条\`}\` 提示，总计信息更清晰。

* 使用 `ResizeObserver` 代替 `resize` 事件，提升高度计算的鲁棒性。
