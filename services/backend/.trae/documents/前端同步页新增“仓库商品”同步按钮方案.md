## 目标
- 在 React 同步页面新增“仓库商品（CBP_DIC_商品库存表 → WarehouseProduct）”的同步按钮与结果展示。

## 变更点
- services：在 `ReactUmi/my-app/src/services/dataSync.ts` 新增 `syncWarehouseProducts()` 方法，调用 `POST /api/react-sync/warehouse-products`。
- 页面：在 `SystemSettings/HqDataSync/index.tsx` 引入方法，新增卡片：
  - 紧凑样式（与现有卡片一致）
  - 按钮触发同步，展示 `SyncResult`（状态/新增/错误/耗时/消息）

## 验证
- 构建通过（`npm run build`）。
- 点击按钮触发同步，页面提示与结果展示正常。

## 影响范围
- 仅前端新增一张卡片与服务方法，不影响其他功能。
