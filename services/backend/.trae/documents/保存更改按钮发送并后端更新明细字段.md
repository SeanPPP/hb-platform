## 原因分析

* 前端“保存更改”当前已发送 `NewAutoRetailPrice`、`PricingFloatRate`、`AutoPricing`、`RetailPrice`、`LastPurchasePrice`，但未包含 `StoreProductCode` 与 `ProductCode`（见 React 详情页保存载荷构造：`ReactUmi/my-app/src/pages/PosAdmin/LocalSupplierInvoiceDetail/index.tsx:979-1003`）。

* 后端批量明细保存 DTO `InvoiceDetailUpsertItemDto` 不含 `StoreProductCode` 与 `ProductCode`（`BlazorApp.Shared/DTOs/LocalSupplierInvoiceReactDtos.cs:166-184`），导致即使前端发送也无法落库。

* 服务方法 `BatchUpsertDetailsAsync` 在更新/插入映射中未设置 `StoreProductCode` 与 `ProductCode`（`blazorapp.api/services/react/LocalSupplierInvoicesReactService.cs:464-487`、`492-520`、更新用 `.SetColumnsIF(...)` 段 `534-589`）。

## 变更方案

* 前端：

  * 扩展类型 `InvoiceDetailUpsertItemDto` 增加 `storeProductCode?: string`、`productCode?: string`（`ReactUmi/my-app/src/types/localSupplierInvoice.ts:131-148`）。

  * 在“保存更改”按钮构造 `upserts` 时加入两字段，并做缺省回填：

    * `productCode: it.productCode || rowMeta[it.detailGUID]?.supplierDetect?.productCode`

    * `storeProductCode: it.storeProductCode || rowMeta[it.detailGUID]?.supplierDetect?.storeProductCode`

    * 修改位置：`ReactUmi/my-app/src/pages/PosAdmin/LocalSupplierInvoiceDetail/index.tsx:979-1005`。

* 后端：

  * 扩展共享 DTO `InvoiceDetailUpsertItemDto` 新增属性：`public string? StoreProductCode { get; set; }`、`public string? ProductCode { get; set; }`（`BlazorApp.Shared/DTOs/LocalSupplierInvoiceReactDtos.cs`）。

  * 服务方法 `BatchUpsertDetailsAsync`：

    * `toUpdate`/`toInsert` 映射补充 `StoreProductCode = it.StoreProductCode`、`ProductCode = it.ProductCode`（`blazorapp.api/services/react/LocalSupplierInvoicesReactService.cs:468-487`、`493-520`）。

    * 更新语句增加：

      * `.SetColumnsIF(u.StoreProductCode != null, x => x.StoreProductCode == u.StoreProductCode)`

      * `.SetColumnsIF(u.ProductCode != null, x => x.ProductCode == u.ProductCode)`

      * 位置：`blazorapp.api/services/react/LocalSupplierInvoicesReactService.cs:534-589` 中相同风格追加。

    * 回落插入 `ins` 同步添加两字段（`blazorapp.api/services/react/LocalSupplierInvoicesReactService.cs:593-619`）。

* 校验：实体已存在两字段（`BlazorApp.Shared/Models/HBweb/StoreLocalSupplierInvoiceDetails.cs:50-65`），长度约束 `Length=50`，保持传值不超过 50。
