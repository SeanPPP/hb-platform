# CartBadge按钮间距优化 - "View Cart"按钮不再拥挤

## 🎯 问题描述

用户反馈"View Cart"按钮有点挤，需要为按钮提供更多的呼吸空间和更好的视觉布局。

## 🔧 优化内容

### 1. 桌面端布局优化

#### 主容器调整
```css
.desktop-cart-summary {
    gap: 8px;               /* 从20px减少到8px，为内容分配空间 */
    padding: 12px 24px;     /* 左右内边距从20px增加到24px */
    min-width: 480px;       /* 新增最小宽度确保足够空间 */
}
```

#### 购物车信息容器
```css
.cart-info {
    gap: 16px;              /* 从20px减少到16px，为按钮让出空间 */
    margin-right: 16px;     /* 新增与按钮区域的间距 */
}
```

#### 按钮容器优化
```css
.cart-actions {
    flex-shrink: 0;         /* 新增：防止按钮区域被压缩 */
    min-width: 100px;       /* 新增：确保按钮有足够的最小宽度 */
}
```

#### 单项元素优化
```css
.cart-item {
    margin: 0;              /* 移除边距，使用父容器的gap */
    gap: 6px;               /* 图标、数值、标签之间的间距从8px减少到6px */
}
```

### 2. 移动端布局优化

#### 移动端容器调整
```css
.mobile-cart-summary {
    gap: 8px;               /* 从12px减少到8px，适应小屏幕 */
    padding: 8px 16px;      /* 左右内边距从12px增加到16px */
}
```

#### 移动端单项优化
```css
.mobile-cart-item {
    margin: 0;              /* 移除边距，使用父容器的gap */
    gap: 3px;               /* 紧凑间距，从4px减少到3px */
}
```

## 🎨 视觉改进

### 间距层级
1. **容器内边距**: 12px 24px (增加左右空间)
2. **内容区与按钮间距**: 16px (清晰分隔)
3. **数据项间距**: 16px (保持可读性)
4. **图标文字间距**: 6px (紧凑但不拥挤)

### 空间分配
- **内容区域**: flex: 1 (占据主要空间)
- **按钮区域**: min-width: 100px (确保最小宽度)
- **整体宽度**: min-width: 480px (桌面端最小宽度)

## 📱 响应式优化

### 桌面端 (≥769px)
- 更大的内边距和间距
- 清晰的内容和按钮分离
- 充足的点击区域

### 移动端 (≤768px)
- 紧凑但不拥挤的布局
- 保持触摸友好的按钮大小
- 优化的小屏幕空间利用
