# UI设计原则 - HB Platform多店铺订单管理系统

## 设计哲学

### 核心理念
**"简洁、高效、专业"**
- 简洁：界面清晰，减少认知负担
- 高效：操作路径最短，提升工作效率  
- 专业：符合B2B企业级应用标准

### 设计目标
1. **降低学习成本**：新用户5分钟内可上手基本操作
2. **提升操作效率**：常用操作不超过3次点击
3. **减少错误操作**：关键操作有明确确认机制
4. **优化视觉体验**：长时间使用不疲劳

## 布局原则

### 响应式断点
```css

--tablet: 768px;      /* 平板端 */
--desktop: 992px;     /* 桌面端 */
--wide: 1200px;       /* 宽屏端 */
```

### 网格系统
#### 12栅格系统
- **移动端**: 单列布局，垂直堆叠
- **平板端**: 2-3列布局，关键信息优先
- **桌面端**: 3-4列布局，充分利用空间
- **宽屏端**: 多列布局，信息密度最大化

## 组件设计规范

### 导航组件

#### 侧边导航 (Sidebar)
- **宽度**: 240px (展开) / 64px (收起)
- **位置**: 左侧固定
- **层级**: 最多3级
- **交互**: 悬停高亮，点击选中
- **图标**: 16px标准图标 + 文字

#### 顶部导航 (Top Bar)
- **高度**: 60px
- **内容**: 页面标题 + 用户操作区
- **固定**: 滚动时保持固定
- **面包屑**: 显示当前位置路径

### 数据展示组件

#### 卡片 (Card)
- **圆角**: 8px
- **阴影**: 默认阴影，悬停增强
- **内边距**: 24px
- **分组**: 相关卡片间距16px

#### 表格 (Table)
- **行高**: 48px (标准) / 40px (紧凑)
- **边框**: 仅水平分割线
- **悬停**: 行悬停高亮
- **排序**: 表头支持排序
- **分页**: 底部固定分页器

#### 列表 (List)
- **项间距**: 16px
- **分隔线**: 1px浅色分割线
- **头像**: 40px圆形头像
- **辅助信息**: 12px次要文字

### 表单组件

#### 输入框 (Input)
- **高度**: 40px (标准) / 32px (紧凑)
- **边框**: 1px，聚焦时2px主色
- **圆角**: 6px
- **内边距**: 12px水平，8px垂直
- **错误状态**: 红色边框 + 错误提示

#### 按钮 (Button)
- **主要按钮**: 主色背景，白色文字
- **次要按钮**: 白色背景，主色边框
- **文字按钮**: 无边框，主色文字
- **禁用状态**: 降低透明度，禁用光标

#### 选择器 (Selector)
- **下拉高度**: 最大300px
- **选项高度**: 40px
- **搜索框**: 顶部固定搜索
- **多选**: 标签式展示已选项

### 反馈组件

#### 消息提示 (Message)
- **位置**: 右上角
- **持续时间**: 3秒 (成功) / 5秒 (错误)
- **类型**: 成功、警告、错误、信息
- **动画**: 滑入滑出

#### 对话框 (Modal)
- **遮罩**: 黑色半透明背景
- **动画**: 淡入淡出 + 缩放
- **按钮**: 右上角关闭 + 底部操作按钮
- **尺寸**: 小(400px) / 中(600px) / 大(800px)

## 交互设计原则

### 1. 操作反馈
- **即时反馈**: 所有操作在200ms内有反馈
- **加载状态**: 骨架屏 + 加载动画
- **成功反馈**: 绿色提示 + 成功音效
- **错误反馈**: 红色提示 + 详细错误信息

### 2. 操作效率
- **快捷键**: 支持键盘快捷键操作
- **批量操作**: 支持多选批量处理
- **快速搜索**: 全局搜索框，支持模糊匹配
- **最近使用**: 记住用户常用操作

### 3. 防错机制
- **确认对话框**: 删除、修改等危险操作需确认
- **撤销操作**: 支持操作撤销和重做
- **自动保存**: 表单输入自动保存草稿
- **输入验证**: 实时输入验证和提示

### 4. 状态管理
- **加载状态**: 骨架屏占位，避免布局跳动
- **空状态**: 友好的空数据提示
- **错误状态**: 清晰的错误信息和解决方案
- **离线状态**: 离线提示和缓存数据展示

## 无障碍设计

### 1. 键盘导航
- **Tab顺序**: 逻辑顺序的Tab键导航
- **焦点指示**: 清晰的焦点指示器
- **快捷键**: 常用操作的键盘快捷键
- **跳过链接**: 提供"跳到主要内容"链接

### 2. 屏幕阅读器支持
- **语义化标签**: 使用语义化HTML标签
- **ARIA标签**: 适当的ARIA属性
- **替代文本**: 所有图片提供alt文本
- **状态描述**: 动态内容更新有屏幕阅读器提示

### 3. 视觉无障碍
- **颜色对比**: 文字与背景对比度至少4.5:1
- **字体大小**: 支持最小12px，推荐14px以上
- **图标标签**: 所有图标都有文字标签
- **错误指示**: 不仅用颜色，还用图标和文字指示错误

## 移动端适配

### 1. 触摸目标
- **最小尺寸**: 44x44px的触摸目标
- **间距**: 触摸目标间最小8px间距
- **手势**: 支持滑动手势操作
- **长按**: 长按显示操作菜单

### 2. 响应式布局
- **断点适配**: 375px、768px、1024px、1440px
- **内容优先级**: 移动端优先显示关键信息
- **导航模式**: 侧边栏转为底部标签栏
- **手势导航**: 支持左右滑动切换页面

### 3. 性能优化
- **图片优化**: 响应式图片，移动端加载小图
- **懒加载**: 滚动加载，减少初始加载时间
- **缓存策略**: 合理使用浏览器缓存
- **离线支持**: 支持PWA离线访问

## 动效设计

### 1. 过渡动画
- **页面切换**: 300ms的淡入淡出
- **状态变化**: 200ms的平滑过渡
- **展开收起**: 250ms的高度动画
- **排序过滤**: 300ms的位置动画

### 2. 微交互
- **悬停效果**: 颜色变化 + 阴影增强
- **点击反馈**: 0.95的缩放效果
- **加载动画**: 骨架屏 + 脉冲效果
- **成功动画**: 绿色勾选 + 淡出效果

### 3. 动画性能
- **GPU加速**: 使用transform和opacity属性
- **避免重排**: 减少影响布局的属性变化
- **节流优化**: 滚动和输入事件节流处理
- **减少层级**: 避免过多的嵌套层级
