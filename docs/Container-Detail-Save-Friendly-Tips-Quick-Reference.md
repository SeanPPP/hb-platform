## 📌 技术要点

### 核心逻辑
```csharp
// 1. 检查是否有商品信息需要更新
var hasProductChanges = changedProducts.Any();

// 2. 显示友好提示
if (hasProductChanges)
    MessageService.Info("开始保存..., 同时将同步更新...");

// 3. 保存明细
var response = await ContainerService.BatchUpdateContainerDetailsAsync(...);

// 4. 同步商品信息（如果有）
if (hasProductChanges)
    var productResult = await SaveProductChangesWithResult();

// 5. 显示综合结果
MessageService.Success("✅ 货柜明细...\n✅ 商品信息...");
```

### 关键方法
- `SaveAllDetails()` - 保存全部明细
- `SaveSelectedDetails()` - 保存选中明细
- `SaveProductChangesWithResult()` - 保存商品信息并返回结果
