# 仓库商品批量管理 - 快速开始指南

## 🚀 5分钟快速上手

### 第一步：访问页面

```
URL: http://localhost:5000/warehouse/batch-edit
权限: 需要 Admin 或 Warehouse 角色
```

### 第二步：过滤商品

1. 点击"过滤条件"展开过滤面板
2. 输入过滤条件（如：商品编码、商品名称等）
3. 点击"搜索"按钮

### 第三步：编辑商品

**方法1：双击编辑**
- 双击任意单元格
- 输入新数据
- 点击其他地方（失焦）自动保存编辑状态

**方法2：Excel粘贴**
1. 在Excel中复制数据（`Ctrl+C`）
2. 在表格区域按 `Ctrl+V`
3. 确认粘贴

### 第四步：保存数据

**单行保存**：
- 点击每行的"保存"按钮

**批量保存**：
- 点击顶部"保存全部修改(N)"按钮
- 或使用快捷键 `Ctrl+S`

---

## 📖 常用操作

### 批量操作

1. 勾选要操作的商品
2. 点击"批量操作"按钮
3. 选择操作类型：
   - 批量设置价格
   - 批量调整库存
   - 批量设置状态
   - 批量设置仓位
4. 填写参数
5. 确认执行

### 仓位编辑

1. 点击商品行的"编辑"按钮
2. 在弹窗中选择新仓位
3. 点击"确定"保存

### 导出数据

1. 点击顶部"导出数据"按钮
2. 选择格式（Excel或PDF）
3. 自动下载文件

## 💡 使用技巧

### 1. 从货柜明细跳转

在货柜明细页面：
```csharp
// 选择商品 → 批量编辑
private void OpenBatchEdit()
{
    var codes = SelectedProducts.Select(p => p.ProductCode).ToList();
    Navigation.NavigateTo($"/warehouse/batch-edit?codes={string.Join(",", codes)}");
}
```

### 2. Excel数据格式

复制Excel时，按以下列顺序：

| 国内价 | 贴牌价 | 进口价 | 库存 | 最小订货量 | 体积 |
|--------|--------|--------|------|-----------|------|
| 50.00  | 60.00  | 70.00  | 100  | 10        | 0.05 |

### 3. 快捷键

- `Ctrl+S`：保存全部修改
- `Ctrl+V`：粘贴Excel数据

### 4. 库存金额自动计算

修改以下字段时，库存金额自动更新：
- 进口价格
- 库存数量

公式：`库存金额 = 库存数量 × 进口价格`

## ⚠️ 注意事项

### 1. 数据并发

- 系统使用乐观锁控制并发
- 如果数据被其他用户修改，保存时会提示"数据已被修改，请刷新"
- 刷新页面后重新编辑

### 2. 未保存提示

- 有未保存的修改时，关闭页面会弹出确认对话框
- 确保不丢失数据

### 3. 仓位限制

- 一个商品只能设置一个仓位（业务规则）
- 设置新仓位会自动删除旧仓位

### 4. 分页说明

- 默认每页显示50条
- 可选：50/100/200/500/1000
- 大数据量时建议使用较小的分页

## 🐛 常见问题

### Q1: 粘贴功能不工作？

**解决方案**：
1. 确保在表格区域按 `Ctrl+V`
2. 检查浏览器是否允许访问剪贴板
3. 尝试刷新页面

### Q2: 保存失败？

**可能原因**：
1. 数据验证失败（价格/库存为负数）
2. 并发冲突（数据已被修改）
3. 网络连接问题

**解决方案**：
1. 检查数据是否符合验证规则
2. 刷新页面重新编辑
3. 检查网络连接

### Q3: 导出功能显示"尚未实现"？

**说明**：
- Excel/PDF导出功能需要后续开发
- 当前版本暂不支持
