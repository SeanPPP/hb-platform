## 🚀 5分钟快速测试流程

### 步骤 1: 准备测试环境（1分钟）
```
✅ 打开货柜明细页面
✅ 确认货柜状态为"已确认"或"草稿"（可编辑状态）
✅ 表格中有商品数据
```

### 步骤 2: 执行商品检测（1分钟）
```
1. 点击页面上的 [检测商品] 按钮
2. 等待检测完成（显示成功提示）
3. 查看表格最右侧的"检测结果"列
```

**预期结果：**
- 🟢 新商品显示：`新商品 ➕`（绿色标签，可点击）
- 🔵 已存在商品显示：`已存在`（蓝色标签）
- 🔵 未检测商品显示：`未检测`（灰蓝色标签）

### 步骤 3: 测试基本创建功能（3分钟）

#### 3.1 测试缺少英文名称的情况 🆕
```
1. 找一个 [新商品 ➕] 标签的商品
2. 检查它的"英文名称"列是否为空
3. 点击 [新商品 ➕] 标签
```

**预期结果：**
```
❌ 错误提示："❌ 商品 XXX 缺少英文名称，请先设置后再创建"
```

#### 3.2 测试缺少进口价格的情况 🆕
```
1. 找一个 [新商品 ➕] 标签的商品（有英文名称）
2. 检查它的"进口价格"列是否为空或"--"
3. 点击 [新商品 ➕] 标签
```

**预期结果：**
```
❌ 错误提示："❌ 商品 XXX 缺少进口价格，请先设置后再创建"
```

#### 3.3 测试缺少贴牌价格的情况
```
1. 找一个 [新商品 ➕] 标签的商品（有英文名称和进口价格）
2. 检查它的"贴牌价格"列是否为空或"--"
3. 点击 [新商品 ➕] 标签
```

**预期结果：**
```
❌ 错误提示："❌ 商品 XXX 缺少贴牌价格，请先设置后再创建"
```

#### 3.4 设置所有必填字段并重试
```
1. 双击该商品的"进口价格"列单元格
2. 输入价格，例如：12.50
3. 按 Enter 键保存
4. 双击该商品的"贴牌价格"列单元格
5. 输入价格，例如：15.99
6. 按 Enter 键保存
7. 再次点击 [新商品 ➕] 标签
```

**预期结果：**
```
✅ 成功提示："✅ 成功创建商品：XXX"
✅ 自动重新检测商品
✅ 该商品标签变为 [已存在]（蓝色）
```

### 步骤 4: 验证数据（1分钟）

#### 4.1 前端验证
```
1. 找到刚创建的商品行
2. 确认"检测结果"列显示 [已存在]
3. 确认价格数据保持不变
```

#### 4.2 后台验证（可选）
```sql
-- 检查商品是否创建成功
SELECT * FROM Product WHERE ItemNumber = '{货号}';

-- 检查仓库商品是否创建
SELECT * FROM WarehouseProduct WHERE ProductCode = '{ProductCode}';

-- 检查店铺零售价是否创建（应该为所有活跃店铺）
SELECT * FROM StoreRetailPrice WHERE ProductCode = '{ProductCode}';
```

## 🐛 常见问题速查

### 问题1：点击"新商品"标签没反应
**原因：** 可能是 JavaScript 错误或网络问题
**解决：**
1. 按 F12 打开浏览器控制台
2. 查看 Console 是否有错误
3. 查看 Network 是否有失败的请求
4. 刷新页面重试

### 问题2：一直提示缺少贴牌价格
**原因：** 价格输入未保存或输入值为0
**解决：**
1. 确认价格输入后按了 Enter 键或点击了其他地方
2. 确认价格显示在单元格中
3. 确认价格 > 0
4. 如果还不行，点击"保存选中明细"按钮

### 问题3：创建后标签还是"新商品"
**原因：** 重新检测未触发
**解决：**
1. 手动点击"检测商品"按钮
2. 查看浏览器控制台日志
3. 检查后台日志确认是否创建成功

## 💡 测试技巧

### 技巧1：快速找到新商品
1. 先点击"检测商品"
2. 在"检测结果"列筛选器中输入"新商品"
3. 表格自动过滤只显示新商品

### 技巧2：批量设置价格
1. 选中多个新商品（勾选复选框）
2. 双击"调整浮率"列单元格，设置浮率（例如1.3）
3. 价格自动计算
4. 点击"保存选中明细"
5. 逐个点击"新商品 ➕"标签创建

### 技巧3：验证创建结果
1. 创建成功后，标签应立即变为"已存在"
2. 如果没变，查看成功提示是否显示
3. 手动点击"检测商品"强制刷新
